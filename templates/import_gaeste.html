Hinweise zum Import
  • Unterstützte Spalten: Vorname, Nachname, Kategorie, Seite, Status, Anzahl_Personen, Kind, Weisser_Saal, Anzahl_Essen, Anzahl_Party, Email, Kontakt, Adresse, Bemerkungen
  • Pflichtfelder: Vorname ist erforderlich
  • Duplikate: Werden basierend auf Vorname + Nachname erkannt und entfernt
  • Teilnahme-Logik: Wird automatisch basierend auf Anzahl-Feldern generiert
  • Arbeitsblätter: Das System erkennt automatisch "Gesamt" und "Einzelaufstellung" Arbeitsblätter
